Myjob - Job Postings HTML5 Template

It is an HTML template that you can use to list your job postings. Here you will find general information about the use of this template. Please read and follow these notes carefully.

  • Created: 03/08/2017
  • Latest update: 03/15/2017
  • by: IslandTheme
  • Email: info@kodyum.com

Features

  • Built with Bootstrap 3.x
  • HTML5 & CSS3
  • 30 verified HTML5 templates
  • 100% responsive and clean design
  • Clean and organized code structure
  • And much more...

File Structure

When you start a job, make your file like this.


    - assets
        - css
        - icons
        - images
        - js
        - sass
        - config.rb
   - index.html

Css Structure

Attachments, fonts and other CSS files @import are included in the page. You can reach it untouched by the assets/sass/app.css way.

SCSS should be used when writing custom CSS codes for this template . The file structure is like this.


    - assets
        - css (Plugin and compiled SCSS/SASS files are found here)
        - sass (SCSS/SASS files are found here)
            - components
            - core (Functions, variables, auxiliary classes are found here)
            - app.scss (You can write your SCSS codes here)
        - config.rb (SCSS configuration file)

SCSS/SASS Configuration

To use Scss and Sass, you can sass-lang.com instructions on the sass-lang.com link.

Compass is used for compilation. How to set up and to use these compass-style.org you can see the link.

Our project assets folder is a project created with Compass. To view and compile Sass and Scss files there you can type this command at the command line and type the following command:

compass watch

Javascript

All Javascript files assets/js/ are located in the folder. The app.js file in this folder is for you to write Javascript codes.

Plugins

All add-ons assets/js/ are located in the files folder. The file is used to activate the app.js plugins.

Add-ons used:

  • Bootstrap slider
  • Owl Carousel
  • Summernote
  • Waves Effect
  • Animate.css
  • Scroll.js
  • Typed.js

All plugin settings can be assets/js/app.js accessed from the file.

Customize

Use assets/sass/app.scss when you need to write custom CSS codes. You can also assets/sass/core/vars.scss customize the template by changing the values ​​of the variables in the file.

Note: You must compile after the changes you made in these files.

Variables

The most commonly used variables are contained in assets/sass/vars.scss this file. In this file you can adjust the basic components such as default theme color, default link color, Header , Section , Footer . Necessary explanations were made in this file.

Helper Classes

There are helpful classes to make your job easier. With these classes you can get rid of inline CSS. You can see all helper assets/sass/utility.scss classes in the file. You can also add your own helper classes to this file.

Default helper class groups:

  • margin m-*
  • margin-top m-t-*
  • margin-bottom m-b-*
  • margin-left m-l-*
  • margin-right m-r-*
  • margin-top & margin-bottom m-t-b-*
  • margin-left & margin-right m-l-r-*
  • padding p-*
  • padding-top p-t-*
  • padding-bottom p-b-*
  • padding-left p-l-*
  • padding-right p-r-*
  • padding-top & padding-bottom p-t-b-*
  • padding-left & padding-right p-l-r-*
  • height h-*
  • font-size f-s-*
